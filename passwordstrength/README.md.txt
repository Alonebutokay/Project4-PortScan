# 🔐 Password Strength Checker

A simple Python GUI app that checks the strength of a password and provides suggestions to improve it.

## 🚀 Features
- Checks for:
  - Length
  - Use of uppercase/lowercase
  - Numbers
  - Special characters
  - Dictionary words (optional via NLTK)
- Gives strength rating (Weak/Moderate/Strong)
- Suggests improvements

## 📦 Requirements
- Python 3
- tkinter (built-in)
- regex (built-in)
- nltk (optional for dictionary detection)

## ▶️ How to Run

```bash
pip install nltk
python password_checker.py
